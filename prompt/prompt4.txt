Project: Document Generation Web Application

Key Requirements:
1. Landing Page Features
    - Free document generation for first-time users
    - Template preview functionality
    - Document search or popular templates display
    - PDF download capability
    - Seamless registration flow

2. User Flows:
    A. New User (Landing Page):
        - View/search templates
        - Preview template
        - Generate document
        - Register to download
        - Automatic download after registration
        - Document appears in history

    B. Direct Registration:
        - Register normally
        - Receive "1 free document" popup
        - Access template library

3. Admin Features:
    - Upload document templates
    - Provide preview template version
    - Set template display format
    - Manage placeholder text

Technical Specifications:
- Backend: FastAPI
- Document Format: PDF
- Template System: Dual template storage (preview + extraction versions)
- User Authentication: Required for downloads
- Storage: Document history per user

Implementation Priority:
1. Landing page template preview system
2. Document generation pipeline
3. User registration + authentication flow
4. Admin template management
5. Document history tracking
